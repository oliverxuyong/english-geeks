| 项目 | 判断 |

| **语种** | 英语（Whisper：`language: "en"`） |

## 建议重点听的词（对照文稿）

以下词在 **General American** 与 **明尼苏达/中西部** 之间常有可听差异；请在对应句子的 `s00N.mp3` 或上方 sample 里反复听。

### 1. 卷舌 r（美式共性）

飞行员 **卷舌清晰**，典型美式：

- **Air Force**（s013）
- **demonstration**（s002）
- **Memorial / weekend**（s004、s011）
- **honor**（s014）— 美式常读 `/ˈɑnər/`，非英式非卷舌 `/ˈɒnə/`

### 2. 元音与中西部可能特征

| 词 | 句 | 听什么 |
|----|-----|--------|
| **about** / **out** | s003, s011 | 中西部有时 `/aʊ/` 略偏 `[əʊ]` 或更单音节；南方则常更明显。此处预期接近标准美式。 |
| **bag** / **back** | （若出现） | 北中部偶见 **æ 抬升**（bag 接近 “beg”）；本课文稿中不突出。 |
| **cot / caught** | — | 明尼苏达常见 **cot-caught 合并**（两词元音接近）；采访体里不易单独测，整体听感仍偏美式中性。 |
| **Minnesota / phenomenal** | s005 | *phenomenal* 重音在第二音节，美式 `/fəˈnɑmənəl/`。 |
| **truthfully** | s004, s009 | 口语填充， Midwest + 军事采访里很常见。 |

### 3. 语法与用词（美式，非口音但印证背景）

- **you guys**（s006、s010）— 美式口语复数「你们」
- **G-Force**（s008）— 美式航空说法
- **go out to the beach**（s011）— 佛罗里达迈阿密语境，美式生活用语
- **quite an honor**（s014）— 美式礼貌收尾，非英式 *quite* 的「很」用法为主

### 4. 本课可听见的转写特点（非口音，供学习注意）

Whisper 转写保留口语：

- **"in pilot"** → 规范应为 **"and pilot"**（s002）
- **"Hyundai Air and C show"** → 应为 **"Air and Sea Show"**（s005）
- **"they're able"** → 应为 **"we're able"**（s004）

这些说明是 **现场口语 + 环境噪音**，不是口音类型问题。
